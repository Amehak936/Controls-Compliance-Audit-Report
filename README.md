## 📖 Overview

This project was completed as part of the **Datacom Cyber Security Operations Job Simulation**. The simulation provides hands-on experience in cybersecurity operations through realistic scenarios involving cyber incidents, risk assessments, threat analysis, and business-focused security recommendations. The program focuses primarily on **Governance, Risk & Compliance (GRC)** and **Threat Intelligence & Analysis**, two critical areas of modern cybersecurity. 

Through this simulation, I worked as a cybersecurity professional investigating security incidents, evaluating organizational risks, identifying vulnerabilities, and developing actionable recommendations to improve an organization's security posture.

# 📚 Project Structure

## 🛑 Task 1: Security Breach Impact Analysis

### Objective
Investigate a ransomware attack against a healthcare technology company and assess its impact on the organization. This task simulates the responsibilities of a cybersecurity analyst responding to a real-world security incident. :contentReference[oaicite:1]{index=1}

### Scenario

**Organization:** Orion Health Services

Orion Health Services is a healthcare technology provider that manages sensitive patient information and provides cloud-based software solutions to clinics throughout Australia and New Zealand. The organization experienced a ransomware attack that disrupted operations and compromised sensitive data. :contentReference[oaicite:2]{index=2}

### Incident Details

| Category | Details |
|-----------|----------|
| Attack Type | 🔴 Ransomware |
| Initial Access | 📧 Phishing Email |
| Entry Method | Malicious Excel Attachment |
| Target User | Finance Team Member |
| Systems Impacted | File Server, HR Systems, Finance Systems |
| Backup Server | Partially Encrypted |

### Compromised Assets

- 👥 Employee Payroll Records
- 📅 Patient Appointment Schedules
- 🔑 Internal System Credentials

### Indicators of Compromise (IOCs)

- 🌎 Suspicious login from an overseas IP address
- 🔓 Credential harvesting using Mimikatz
- 🔒 Files encrypted with `.orionlock` extension

### Key Learning Outcomes

This task focused on:

- Identifying attack vectors
- Understanding ransomware behavior
- Evaluating business impact
- Assessing legal and reputational consequences
- Developing incident response recommendations

### Objective

Conduct a comprehensive cybersecurity risk assessment for a large retail organization and identify risks before they become active security incidents. :contentReference[oaicite:3]{index=3}

**RetailNova Pty Ltd**
